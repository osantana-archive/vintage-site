Porquê Linux?

Esse livro não é só para Linux, mas vocês verão em diversos trechos deste texto que eu faço referências ao mesmo. É apenas o sistema que eu uso diariamente por muito tempo e no qual eu desenvolvo os meus joguinhos. Não é nada pessoal, apenas uma preferência pessoal. Para os usuários de outros sistemas, eu tentarei explicar algumas coisas no site http://www.desenvjogos.net onde existirão links para listas de discussões.

Linux também foi escolhido por ser um sistema que barateia os custos de aprendizado pois disponibiliza um sistema completo de desenvolvimento a custo quase zero. Mas fiquem a vontade para me enviarem sugestões, exemplos e correções para osvaldo@desenvjogos.com.

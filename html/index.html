Desenvolvimento de Jogos

Esse site foi planejado para futuramente hospedar os recursos do livro sobre desenvolvimento de jogos que estou escrevendo. Entretanto disponho atualmente de 500MB de espaço para hospedagem e não usarei isso. Resolvi então compartilhar esses recursos com desenvolvedores de jogos brasileiros.

Para você poder utilizar esses recursos você tem que atender algumas regras.

Regras

  • Hospedar apenas jogos;
  • Os jogos devem ter obrigatoriamente os fontes disponíveis para download;
  • As páginas não devem ter: Framesets, Java, Javascript, Flash, realaudio ou qualquer outra tecnologia comercial que seja incompatível com browsers diversos;
  • Não serão toleradas propagandas de qualquer tipo, exceto, se provado que a propaganda é de algum patrocinador.

Vantagens

  • Link rápido
  • Endereço fácil: http://www.desenvjogos.net/seu_jogo
  • Sem banners, propagandas ou outras brincadeiras de mau gosto

Desvantagens

  • Por não ter um esquema melhor, todo o conteúdo da página deve ser enviado para mim comprimido com ZIP ou tar.{gz,bz2} pelo endereço: acidbase@bigfoot.com. Lembre-se que já estou organizando uma solução mais "elegante" que esta.
  • Quota de 5MB de espaço (pretendo aumentar isso pra 10MB).

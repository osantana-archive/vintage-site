O livro que usamos é o Como programar em C de Deitel. E eu recomendo o mesmo
para o aprendizado de programação usando Linguagem C. Para quem já sabe
programar recomendo a leitura de outros livros:

   - C - A linguagem de programação padrão ANSI
         Um clássico escrito pelos criadores da linguagem C. A tradução não é
         das melhores, mas não compromete a boa qualidade do livro que serve
         basicamente para quem já sabe programar.

   - C Completo e Total
         Mais detalhado que o livro acima, é um bom livro para pessoas que
         já programam. A tradução é razoável.

   - GNU C Library
         Documentação (em inglês) das funções da implementação GNU da biblioteca
         ANSI C.

   - A Linguagem C++
         Mais um clássico escrito pelo próprio criador da linguagem. A 
         excelente tradução foi feita pela editora Bookman (que particularmente
         gosto).

   - Programando com Ferramentas GNU
         Livro que explica como usar o ambiente de desenvolvimento GNU
         (infelizmente eles usam Emacs :/). Participei da revisão da tradução
         de parte desse livro.

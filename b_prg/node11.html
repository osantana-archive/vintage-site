Compilando e instalando a SDL

Para usuários de sistemas Linux, faça o login como root e execute: 

# tar xvf SDL-versão.tar.gz | gunzip -c
# cd SDL-versão
# ./configure
# make
# make install

Se tudo estiver correto a SDL já deve estar instalada. Para usuários de distribuições Linux com sistema de empacotamento, é só baixar os pacotes no site http://www.desenvjogos.net e seguir as instruções que estão na página.

Para usuários de outros sistemas operacionais (como o Windows), recomendo uma visita na página da SDL para obter informações de como compilar a mesma.
